## INICIO RÁPIDO

### Desarrollo Local:
```bash
git clone https://github.com/m96792177-hub/Miunave_app.git
cd Miunave_app
npm install

# Terminal 1 - Backend
npm run backend

# Terminal 2 - Frontend  
npm run dev
```

### Scripts Disponibles:
- `npm run dev` - Frontend (puerto 5173)
- `npm run backend` - Backend (puerto 4000)
- `npm run build` - Build producción
- `npm run seed` - Usuario demo
